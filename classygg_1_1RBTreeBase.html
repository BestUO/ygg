Detailed Description

template<class Node, class NB, class NodeTraits, int Tag = 0, class Compare = std::less<Node>>
class ygg::RBTreeBase< Node, NB, NodeTraits, Tag, Compare >

The non-specialized parts of the Red-Black Tree.

This is the main Red-Black Tree class. For more information, see RBTree. This class cannot be instantiated on its own.
