template<class AggValueT, class... Combiners>
class ygg::CombinerPack< AggValueT, Combiners >

This class represents the pack of combiners associated with every node of a Dynamic Segment Tree.

A DynamicSegmentTree can have multiple combiners associated with each node. See TODO for details. Every combiner allows to retrieve a different combined metric (such as maximum, minimum, â€¦) of the aggregate values over arbitrary ranges in the Dynamic Segment Tree.
